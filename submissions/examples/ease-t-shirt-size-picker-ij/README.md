# ease-t-shirt-size-picker

form — t-shirt-size-picker CSS animation component.

## Usage
Open demo.html in a browser. Click the toggle button to see the animation.

## Custom Properties
| Property | Default | Description |
|----------|---------|-------------|
| --primary | #0891b2 | Primary accent color |
| --bg | #f8fafc | Background |
| --duration | 2.9s | Animation duration |

## Notes
CSS handles visual transitions. JavaScript toggles state and manages interaction.
